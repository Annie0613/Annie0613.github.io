# Rock-Paper-Scissors game
A simple interactive Rock, Paper, Scissors game where the user plays against the computer. This project can be implemented in Python, JavaScript, or as a web application.

## Table of Contents
1. [Description](#description)
2. [How to Play](#how-to-play)
3. [Rules](#rules)

## Description
This project is a CLI (Command Line Interface) or Web-based game that allows a human player to compete against a computer-generated choice. The goal is to win the best out of 3, 5, or 9 rounds.

## How to Play
1.  **Start the game:** Run the python script or open the HTML file.
2.  **Make a choice:** Select either `rock`, `paper`, or `scissors`.
3.  **Computer Turn:** The computer will randomly select its move.
4.  **Result:** The winner of the round is displayed, and the score is updated.

## Rules
*   **Rock** beats Scissors.
*   **Paper** beats Rock.
*   **Scissors** beats Paper.
*   If both choices are the same, it is a **Draw**.
